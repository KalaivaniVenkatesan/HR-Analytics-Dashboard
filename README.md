 HR Analytics Dashboard

## Project Overview

This project analyzes employee data to understand attrition, job satisfaction, and workforce trends.

## Tools Used

* Power BI
* Tableau
* Excel
* PostgreSQL (SQL)

##  Key Insights

* Attrition rate analysis
* Department-wise trends
* Job satisfaction heatmap
* Gender and age distribution
